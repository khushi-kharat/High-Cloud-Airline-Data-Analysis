Project Overview
This project analyzes airline operations data for High Cloud Airline to identify trends, improve performance, and support business decision-making.
 Objectives
- Analyze flight performance and delays
- Understand passenger trends
- Identify revenue and route efficiency
- Provide insights using dashboards and SQL queries
- Tools

 Excel
- SQL
- Tablue
- Power BI
Conclusion
This analysis helps the airline understand operational efficiency and make data-driven decisions.
